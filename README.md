# cli_tools
Dotfiles that i use for vim and tmux and oh-my-zsh

Install the dotfiles
```bash
bash install.sh
source install.sh
install_vimrc
```

Update the configuration files
```bash
bash update.sh
```


Uninstall dotfiles that also make a backup
```bash
bash uninstall.sh
source uninstall.sh
uninstall_vimrc
```
